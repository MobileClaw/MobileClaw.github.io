Installation安装指南

How to Install an APK on an Android Phone如何在 Android 手机上安装 APK

For first-time users适合首次使用 About 5 minutes约 5 分钟

If you have not installed an APK manually before, do not worry. The process is simple: download the file, allow installation from your browser or file manager, install the app, and then open it normally.如果你之前没有手动安装过 APK，也不用担心。整个流程很简单：下载文件、允许浏览器或文件管理器进行安装、完成安装，然后正常打开应用即可。

1. Download the APK1. 下载 APK

Open the MobileClaw website on your Android phone and tap the download button. You can also directly download `MobileClaw-release.apk` from the homepage.在 Android 手机上打开 MobileClaw 官网，点击下载按钮即可。你也可以直接从首页下载 `MobileClaw-release.apk`。

2. Find the downloaded file2. 找到下载完成的文件

After the download completes, open the system notification, browser download list, or file manager. In most cases the APK will appear in the `Download` folder.下载完成后，可以从系统通知、浏览器下载列表或文件管理器中打开文件。大多数情况下，APK 会出现在 `Download` 文件夹中。

3. Allow installation if Android asks3. 如果系统提示，请允许安装

Some Android devices block installation from unknown sources by default. If you see a warning, allow installation for the browser or file manager you are using. This setting usually appears only when you first try to install the APK.部分 Android 设备默认会阻止未知来源安装。如果你看到提示，请为当前使用的浏览器或文件管理器临时开启安装权限。通常只有第一次安装 APK 时会看到这个设置。

  • Read the system prompt carefully and make sure the file comes from the official MobileClaw site.请仔细阅读系统提示，并确认文件来自官方 MobileClaw 站点。
  • Do not enable broad installation permissions for apps you do not trust.不要为不信任的应用长期开启大范围安装权限。

4. Complete installation and open the app4. 完成安装并打开应用

Tap `Install`, wait for Android to finish, and then open MobileClaw. On first launch, review the app prompts carefully and continue with the required permission setup.点击“安装”，等待系统完成处理，然后打开 MobileClaw。首次启动时，请认真查看应用内提示，并继续完成必要的权限设置。

5. Recommended next steps5. 推荐的后续步骤

  1. Use a dedicated secondary phone instead of your primary device.优先使用备用设备，而不是主力手机。
  2. Review the safety guidance before enabling powerful permissions.在开启高权限能力前，先阅读安全说明。
  3. Return to the homepage for download and quick-start links whenever needed.如需重新下载安装或查看上手路径，可随时返回首页。
